Bienvenido a Logística Global Ltda.

Esta plataforma permite gestionar vehículos, aeronaves, conductores, pilotos, clientes, cargas, rutas y despachos de manera centralizada. Utiliza los accesos en la navegación superior para explorar los recursos disponibles.

Recursos disponibles

  • API REST: Visita la documentación interactiva para conocer los endpoints disponibles.
  • Autenticación: Obtén un token JWT desde /api/token/ y renuévalo en /api/token/refresh/ para realizar operaciones seguras.
  • Salud del sistema: Comprueba la disponibilidad en /api/ping/.

Para comenzar, inicia sesión en el panel de administración o consume la API según tus necesidades.
